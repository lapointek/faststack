# FastStack

FastStack is a AI generated choose your own adventure story application using Python for the backend and Javascript for the frontend. Containerised using Podman and Docker.

## Installation

### From source

Clone the repository:

```
git clone https://github.com/lapointek/faststack
cd faststack
```

## Usage

### Using podman-compose

```
cd faststack
podman-compose up --build
```

### Using docker-compose

```
cd faststack
docker-compose up --build
```

## About

This project is based on a tutorial implementation [link](https://www.youtube.com/watch?v=_1P0Uqk50Ps) and has been adapted to run in docker containers.
